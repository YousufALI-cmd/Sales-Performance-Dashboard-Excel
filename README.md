# Sales-Performance-Dashboard-Excel
Excel dashboard analyzing sales, profit, and product performance with interactive filters and KPI insights.

## 📌 Overview

An interactive Excel dashboard designed to analyze sales and profitability across regions, products, and shipping modes, enabling quick business insights.

## 🛠 Tools Used

* Microsoft Excel (Pivot Tables, Charts, Conditional Formatting)

## 📊 Key Metrics

* Sales and Profit Analysis
* Shipment Mode Distribution
* Top Profitable States and Cities
* Least Profitable Products

## 🔍 Key Features

* Interactive filters for customer segment, product category, and shipping mode
* KPI-driven visuals for quick performance tracking
* Identification of high and low performing products
* Clean and structured dashboard layout

## 📈 Insights

* Certain states contribute significantly to overall profit
* Some products consistently generate losses
* Shipping modes impact sales distribution
* Customer segments show varying contribution to revenue
